The "Haunted Forest VR" project is a 3D first-person shooter game set in a dark and
eerie forest filled with various monsters. The objective of the game is to locate and
destroy a randomly spawned object within a time limit of 60 seconds while fending off
attacks from enemies. The game was developed as a project for the “3D Modelling and
mixed reality applications” course, aiming to provide an immersive and thrilling VR
experience.

## Implementation

### Technologies Used
- **Unity** - Game development and core gameplay systems  
- **Blender** - 3D modelling and animation

### XR Features
The project uses **Unity’s XR Interaction Toolkit** to implement VR functionality, including:
- Hand interactions
- Player movement
- Object interaction systems

### Gameplay Mechanics
Players explore a haunted forest environment using VR controllers while searching for a target object hidden among hostile creatures. The gameplay combines **exploration, combat, and a time-based challenge**.
